# SAS 焦虑自评量表 & SDS 抑郁自评量表

## SAS 焦虑自评量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 焦虑自评量表（Self-Rating Anxiety Scale） |
| 缩写 | SAS |
| 编制者 | 美国华裔教授 Zung W.W.K.（1971年） |
| 中文版 | 吴文源译制 |
| 适用对象 | 初中生至成人（≥12岁，建议15岁以上） |
| 题目数量 | 20题 |
| 答题形式 | 四级评分 |
| 测试时间 | 约5-10分钟 |
| 用途 | 评估最近一周内焦虑症状的严重程度 |

### 完整题目（20题）

> 请根据**最近一周**内您的实际感受，选择最符合的选项。

| 题号 | 题目内容 | 计分方向 |
|-----|---------|---------|
| 1 | 我觉得比平时容易紧张或着急（焦虑） | 正向 |
| 2 | 我无缘无故地感到害怕 | 正向 |
| 3 | 我容易心里烦乱或感到惊恐 | 正向 |
| 4 | 我觉得我可能将要发疯 | 正向 |
| **5** | **我觉得一切都很好，也不会发生什么不幸** | **反向** |
| 6 | 我手脚发抖打颤 | 正向 |
| 7 | 我因为头疼、颈痛和背痛而苦恼 | 正向 |
| 8 | 我感到容易衰弱和疲乏 | 正向 |
| **9** | **我觉得心平气和，并且容易安静坐着** | **反向** |
| 10 | 我觉得心跳得很快 | 正向 |
| 11 | 我因一阵阵头晕而苦恼 | 正向 |
| 12 | 我有过晕倒发作或觉得要晕倒似的 | 正向 |
| **13** | **我呼气吸气都感到很容易** | **反向** |
| 14 | 我手脚麻木和刺痛 | 正向 |
| 15 | 我因胃痛和消化不良而苦恼 | 正向 |
| 16 | 我常常要小便 | 正向 |
| **17** | **我的手常常是干燥温暖的** | **反向** |
| 18 | 我脸红发热 | 正向 |
| **19** | **我容易入睡并且一夜睡得很好** | **反向** |
| 20 | 我做恶梦 | 正向 |

### 评分标准

#### 各题评分方式

**正向计分题（第1、2、3、4、6、7、8、10、11、12、14、15、16、18、20题）：**

| 选项 | 频率 | 得分 |
|-----|-----|-----|
| A（没有或很少时间） | 少于1天 | 1分 |
| B（有时） | 1-2天 | 2分 |
| C（大部分时间） | 3-4天 | 3分 |
| D（绝大部分或全部时间） | 5-7天 | 4分 |

**反向计分题（第5、9、13、17、19题）：**

| 选项 | 频率 | 得分 |
|-----|-----|-----|
| A（没有或很少时间） | 少于1天 | **4分** |
| B（有时） | 1-2天 | **3分** |
| C（大部分时间） | 3-4天 | **2分** |
| D（绝大部分或全部时间） | 5-7天 | **1分** |

#### 计分方法

**粗分** = 20道题得分之和（范围20-80分）

**标准分** = 粗分 × 1.25（取整数，范围25-100分）

#### 结果解释

| 标准分范围 | 焦虑程度 | 说明 |
|---------|---------|-----|
| **< 50分** | 无显著焦虑 | 正常范围 |
| **50-59分** | 轻度焦虑 | 有一定焦虑症状 |
| **60-69分** | 中度焦虑 | 焦虑症状明显 |
| **≥ 70分** | 重度焦虑 | 严重焦虑，需专业干预 |

## SDS 抑郁自评量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 抑郁自评量表（Self-Rating Depression Scale） |
| 缩写 | SDS |
| 编制者 | 美国精神科医生 Zung W.W.K.（1965年） |
| 中文版 | 吴文源译制 |
| 适用对象 | 初中生至成人（≥12岁，建议15岁以上） |
| 题目数量 | 20题 |
| 答题形式 | 四级评分 |
| 测试时间 | 约5-10分钟 |
| 用途 | 评估最近一周内抑郁症状的严重程度 |

### 完整题目（20题）

> 请根据**最近一周**内您的实际感受，选择最符合的选项。

| 题号 | 题目内容 | 计分方向 |
|-----|---------|---------|
| 1 | 我觉得闷闷不乐，情绪低沉 | 正向 |
| **2** | **我觉得一天之中早晨最好** | **反向** |
| 3 | 我一阵阵哭出来或觉得想哭 | 正向 |
| 4 | 我晚上睡眠不好 | 正向 |
| **5** | **我吃得跟平时一样多** | **反向** |
| **6** | **我与异性密切接触时和以往一样感到愉快** | **反向** |
| 7 | 我发觉我的体重在下降 | 正向 |
| 8 | 我有便秘的苦恼 | 正向 |
| 9 | 我心跳比平时快 | 正向 |
| 10 | 我无缘无故感到疲乏 | 正向 |
| **11** | **我的头脑跟平时一样清楚** | **反向** |
| **12** | **我觉得经常做的事情并没有困难** | **反向** |
| 13 | 我觉得不安而平静不下来 | 正向 |
| **14** | **我对将来抱有希望** | **反向** |
| 15 | 我比平时更容易生气激动 | 正向 |
| **16** | **我觉得做出决定是容易的** | **反向** |
| **17** | **我觉得自己是个有用的人，有人需要我** | **反向** |
| **18** | **我的生活过得很有意思** | **反向** |
| 19 | 我认为如果我死了别人会生活得更好 | 正向 |
| **20** | **平常感兴趣的事我仍然照样感兴趣** | **反向** |

> 注：反向题目（粗体标注）共10题：第2、5、6、11、12、14、16、17、18、20题

### 评分标准

#### 各题评分方式

**正向计分题（第1、3、4、7、8、9、10、13、15、19题）：**

| 选项 | 频率 | 得分 |
|-----|-----|-----|
| A（没有或很少时间） | 少于1天 | 1分 |
| B（有时） | 1-2天 | 2分 |
| C（相当多时间） | 3-4天 | 3分 |
| D（绝大部分或全部时间） | 5-7天 | 4分 |

**反向计分题（第2、5、6、11、12、14、16、17、18、20题）：**

| 选项 | 频率 | 得分 |
|-----|-----|-----|
| A（没有或很少时间） | 少于1天 | **4分** |
| B（有时） | 1-2天 | **3分** |
| C（相当多时间） | 3-4天 | **2分** |
| D（绝大部分或全部时间） | 5-7天 | **1分** |

#### 计分方法

**粗分** = 20道题得分之和（范围20-80分）

**标准分** = 粗分 × 1.25（取整数，范围25-100分）

#### 结果解释

| 标准分范围 | 抑郁程度 | 说明 |
|---------|---------|-----|
| **< 50分** | 无抑郁 | 正常范围 |
| **50-60分** | 轻度抑郁 | 有一定抑郁症状 |
| **61-70分** | 中度抑郁 | 抑郁症状明显 |
| **> 70分** | 重度抑郁 | 严重抑郁，需专业干预 |

---

## SAS 与 SDS 联合使用

两个量表常同时使用，可同时评估焦虑和抑郁：
- SAS 侧重**焦虑症状**（紧张、恐惧、躯体反应）
- SDS 侧重**抑郁症状**（情绪低落、兴趣丧失、自我否定）

---

## 注意事项

1. 两个量表均为**筛查工具**，结果不能替代临床诊断
2. 建议年龄15岁以上使用，12-15岁使用时需关注理解能力
3. 结果的解释需结合学生实际情况和临床观察
4. **SDS第19题涉及自杀意念**，若得分3-4分，需立即评估安全性
5. 量表得分的纵向变化（前后比较）比单次测量更有参考价值
