# SCARED 儿童焦虑性情绪障碍筛查表

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 儿童焦虑性情绪障碍筛查表（Screen for Child Anxiety Related Emotional Disorders） |
| 缩写 | SCARED |
| 编制者 | Birmaher等（1997年编制，1999年修订为41条版本） |
| 中文版修订 | 中南大学湘雅二院 苏林雁、王凯等（2008年修订，制定全国常模） |
| 适用对象 | 6-18岁儿童及青少年 |
| 题目数量 | 41题 |
| 填写方式 | 学生自评（亦可由父母填写） |
| 测试时间 | 约10-15分钟 |
| 用途 | 筛查儿童焦虑相关情绪障碍 |

## 评分标准

### 各题评分方式（三级评分）

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 0 | 没有此问题 | 0分 |
| 1 | 有时有此问题 | 1分 |
| 2 | 经常有此问题 | 2分 |

### 计分方法

**各分量表得分** = 该分量表所有题目得分之和

**总分** = 41道题得分之和（满分82分）

### 总分筛查标准

| 年龄及性别 | 焦虑阳性阈值（总分） |
|----------|------------------|
| 8-11岁（不分性别） | ≥ 23分 |
| 12-18岁男生 | ≥ 22分 |
| 12-18岁女生 | ≥ 24分 |

> 说明：总分达到阈值提示存在焦虑相关情绪问题，建议进一步评估。

### 各分量表筛查标准

| 分量表 | 筛查阳性阈值 | 可能对应诊断 |
|-------|-----------|------------|
| 躯体化/惊恐 | ≥ 7分 | 惊恐障碍 |
| 广泛性焦虑 | ≥ 9分 | 广泛性焦虑障碍 |
| 分离性焦虑 | ≥ 5分 | 分离焦虑障碍 |
| 社交恐怖 | ≥ 8分 | 社交焦虑障碍 |
| 学校恐怖 | ≥ 3分 | 学校恐惧症 |

---

## 注意事项

1. 本量表为**筛查工具**，阳性结果需由专业人员进一步评估
2. 可有效区分焦虑与抑郁症状，避免混淆
3. 对于儿童，建议同时收集父母版以交叉验证
4. 量表对治疗效果敏感，可用于追踪评估
