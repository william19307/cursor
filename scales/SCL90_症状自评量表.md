# SCL-90 症状自评量表

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 症状自评量表（Symptom Checklist-90，又名90项症状清单） |
| 缩写 | SCL-90 |
| 编制者 | L.R. Derogatis（1975年） |
| 中文版 | 吴文源等修订，国内广泛使用 |
| 适用对象 | 初中生至成人（14岁以上，建议高中生及以上） |
| 题目数量 | 90题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约20-30分钟 |
| 用途 | 综合评估心理健康状况和各类精神症状 |

## 评分标准

### 各题评分方式（五级评分）

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 1 | 没有（从无此感觉） | 1分 |
| 2 | 轻度（有一点，较少） | 2分 |
| 3 | 中度（有些，有时） | 3分 |
| 4 | 相当重（常常如此） | 4分 |
| 5 | 严重（几乎一直如此） | 5分 |

> 注：部分版本采用0-4计分，使用时需注意版本差异。

### 统计指标

| 指标 | 计算方法 | 说明 |
|-----|---------|-----|
| **总分** | 90项得分之和 | 反映整体心理健康水平 |
| **总均分** | 总分 ÷ 90 | 反映整体症状严重程度 |
| **阳性项目数** | 单项得分 ≥ 2 的项目数 | 有"症状"的项目数量 |
| **阴性项目数** | 单项得分 = 1 的项目数 | 无"症状"的项目数量 |
| **阳性症状均分** | 总分 ÷ 阳性项目数 | 有问题项目的平均严重程度 |
| **因子分** | 因子各项得分之和 ÷ 因子项目数 | 各维度的症状程度 |

### 全量表结果解释

| 总分 | 心理健康状况 | 建议处理 |
|-----|------------|--------|
| **< 160分** | 正常 | 常规关注 |
| **160-200分** | 有一定心理问题 | 需要关注，适时疏导 |
| **> 200分** | 存在较明显心理障碍 | 需要专业干预 |

### 因子分解释
| 因子均分 | 含义 |
|---------|-----|
| **< 2分** | 该方面无明显问题 |
| **2-3分** | 该方面有轻度问题 |
| **3-4分** | 该方面有中度问题 |
| **> 4分** | 该方面有严重问题 |

**阳性判定标准**：任一因子得分 ≥ 2分为阳性，提示该方面可能存在心理问题。

## 注意事项

1. SCL-90 适用于 **14岁以上**学生，建议高中生及以上使用
2. 对于初中生，建议优先考虑 MHT 或 MSSMHS
3. 结果仅供参考，不能作为临床诊断依据
4. **第15题（自杀意念）和第59题（死亡想法）** 需特别关注
5. 测试前应向学生说明保密原则，确保真实作答
