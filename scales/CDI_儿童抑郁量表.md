# CDI 儿童抑郁量表

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 儿童抑郁量表（Children's Depression Inventory） |
| 缩写 | CDI |
| 编制者 | Kovacs M.（1977年编制，1992年修订） |
| 中文版 | 已在中国大陆标准化，重庆医科大学等机构验证 |
| 适用对象 | 7-17岁儿童及青少年 |
| 题目数量 | 27题 |
| 答题形式 | 三选一（每题3个陈述选最符合的） |
| 测试时间 | 约15-20分钟 |
| 用途 | 评估儿童及青少年抑郁症状的严重程度 |

---

## 量表结构（5个因子）

| 因子 | 对应题目 | 内容 |
|-----|---------|-----|
| 负性情绪/心境 | 1、3、5、8、9、10 | 悲伤、易激惹、自杀意念等 |
| 行为问题 | 11、12、13、14、15、16 | 行为退缩、破坏性行为 |
| 躯体不适 | 17、18、19、20 | 睡眠、饮食、疲劳等 |
| 自我评价 | 2、6、7、21、22、23 | 自尊、自我效能感 |
| 人际问题 | 24、25、26、27 | 人际关系、社交退缩 |

---

## 完整题目（27题）

> 说明：每题列出3个陈述（A/B/C），选择**最近两周**内最符合自己的陈述。

**第1题**
- A（0分）：我偶尔感到难过

- C（2分）：我总是感到难过

## 评分标准

### 各题计分

- **正向计分题**：选A得0分，选B得1分，选C得2分
- **反向计分题**（第7、10、11、13题）：按相反顺序计分

### 总分计算

**总分** = 27道题得分之和（满分54分）

### 结果解释（总分判断标准）

| 年龄段 | 提示抑郁症状阈值 |
|-------|--------------|
| **6-8岁** | 总分 ≥ 13分 |
| **9-12岁** | 总分 ≥ 17分 |
| **12岁以上** | 总分 ≥ 19分（可疑） |
| **通用** | 总分 ≥ 20分（建议临床评估） |

### 严重程度参考
| 总分 | 严重程度 |
|-----|---------|
| 0-9分 | 无抑郁 |
| 10-19分 | 轻度抑郁倾向 |
| 20-29分 | 中度抑郁倾向 |
| ≥ 30分 | 重度抑郁倾向 |

> **特别注意**：第9题（自杀意念）和第22题（自伤想法）若选B或C，无论总分高低，均需立即关注和专业评估。

| 指标 | 数值 |
|-----|-----|
| 内部一致性（Cronbach's α）| 0.83 |
| 重测信度（2周） | 约0.70 |
| 与DSM标准的一致性 | 较好 |

---

## 注意事项

1. 本量表为**筛查工具**，不能替代专业诊断
2. 高分学生需由精神科医生依据DSM标准进行确诊
3. **自杀/自伤相关题目须特别关注**，出现阳性应立即介入
4. 男女生、不同年龄段常模有所差异，需参考标准化常模
