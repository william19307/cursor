## 二、按学段推荐使用量表

### 小学阶段（6-12岁）

| 推荐量表 | 推荐理由 | 特别说明 |
|---------|---------|---------|
| **MHT（小四以上）** | 国内最常用，题目贴近小学生理解水平 | 适合四年级以上 |
| **SCARED** | 有效筛查各类儿童焦虑障碍 | 父母版可辅助低年级使用 |
| **CDI** | 专为儿童设计，抑郁筛查准确性高 | 7岁以上均可使用 |
| **SDQ** | 简短高效，父母/教师均可填写 | 多版本多角度评估 |
| **CBCL** | 行为问题全面评估 | 主要由父母填写 |

### 初中阶段（12-15岁）

| 推荐量表 | 推荐理由 | 特别说明 |
|---------|---------|---------|
| **MHT** | 国内最权威的初中生心理筛查 | 仍然适用 |
| **MSSMHS（MMHI-60）** | 中国本土编制，10维度全面评估 | 初中生最适用 |
| **SCARED** | 焦虑筛查高效准确 | 特别适合初中生 |
| **CDI** | 抑郁筛查 | 13岁以上使用更准确 |
| **PHQ-9 / GAD-7** | 国际通用，简便快速 | 结合中国常模使用 |
| **SAS / SDS** | 成熟量表，广泛使用 | 建议15岁以上使用 |

### 高中阶段（15-18岁）

| 推荐量表 | 推荐理由 | 特别说明 |
|---------|---------|---------|
| **MSSMHS（MMHI-60）** | 专为中学生设计，高中仍适用 | 最推荐 |
| **SCL-90** | 全面系统评估 | 14岁以上可用 |
| **SAS / SDS** | 成熟度高，结果解读简便 | 高中最适用 |
| **PHQ-9 / GAD-7** | 快速筛查抑郁和焦虑 | 国际通用 |

## 五、重要注意事项

### 使用原则

1. **筛查不等于诊断**：所有量表均为筛查工具，阳性结果需由专业人员进一步评估
2. **多源信息**：建议结合学生自评、家长评估、教师观察综合判断
3. **动态追踪**：单次测量的意义有限，建议定期重复测量观察变化趋势
4. **文化适用性**：优先使用经中国常模标准化的量表版本

### 危机信号处理

遇到以下情况，需立即启动危机干预：
- CDI 第9、22题得分 ≥ 1分（自杀/自伤想法）
- PHQ-9 第9题得分 ≥ 1分（自杀/自伤念头）
- SDS 第19题得分 ≥ 3分（死亡想法）
- SCL-90 第15题得分 ≥ 3分（结束生命想法）
- CBCL 第91题得分 ≥ 1分（谈论自杀）

### 保密原则

- 测验结果属于学生个人隐私，不得公开
- 数据存储需符合个人信息保护相关规定
- 向家长反馈时应注意方式方法，避免加重学生心理负担
