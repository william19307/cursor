# CBCL 儿童行为量表（Achenbach系统）

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Achenbach儿童行为量表（Child Behavior Checklist） |
| 缩写 | CBCL |
| 编制者 | 美国心理学家 Achenbach TM（1980年首次编制，多次修订） |
| 中文版 | 上海精神卫生中心等机构引进，1992年完成中国标准化 |
| 适用对象 | 4-16岁儿童（分两个年龄段版本） |
| 题目数量 | 社会能力（20项）+ 行为问题（113项） |
| 版本类型 | 父母（主要评估者）填写 |
| 测试时间 | 约20-30分钟 |
| 用途 | 全面评估儿童社会能力和行为问题 |

---

## 量表结构

### 第一部分：一般项目
儿童基本信息（姓名、性别、年龄、年级、民族等）及主要照料者信息。

### 第二部分：社会能力（7个方面，20个项目）

| 方面 | 评估内容 |
|-----|---------|
| 体育运动 | 参与体育活动的种类和水平 |
| 课外活动 | 兴趣爱好、特长活动 |
| 组织参与 | 参加社会团体、组织的情况 |
| 家务劳动 | 承担家务的情况 |
| 与朋友交往 | 结交朋友数量和交往频率 |
| 人际相处 | 与兄弟姐妹及其他孩子的关系 |
| 学校表现 | 学业成绩及行为表现 |

### 第三部分：行为问题（113项）

按年龄分为两个版本：

| 版本 | 适用年龄 | 主要因子 |
|-----|---------|---------|
| 4-11岁版 | 4-11岁儿童 | 见下表 |
| 12-16岁版 | 12-16岁青少年 | 见下表 |

#### 4-11岁版行为问题因子

| 因子名称 | 包含内容 |
|---------|---------|
| 退缩 | 社交退缩、孤僻行为 |
| 躯体主诉 | 无器质性原因的躯体不适 |
| 焦虑/抑郁 | 焦虑、抑郁情绪 |
| 社会问题 | 与同伴相处的困难 |
| 思维问题 | 奇怪思想、强迫行为 |
| 注意问题 | 注意力不集中、多动 |
| 违规行为 | 破坏规则的行为 |
| 攻击行为 | 言语和身体攻击 |

#### 12-16岁版行为问题因子

| 因子名称 | 包含内容 |
|---------|---------|
| 退缩 | 社交退缩 |
| 躯体主诉 | 躯体化症状 |
| 焦虑/抑郁 | 情绪问题 |
| 社会问题 | 人际困难 |
| 思维问题 | 强迫及奇怪思维 |
| 注意问题 | 注意力问题 |

## 评分标准

### 行为问题评分方式

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 0 | 不符合（在过去6个月无此现象） | 0分 |
| 1 | 偶尔符合（有时有此现象） | 1分 |
| 2 | 完全符合（经常或非常明显） | 2分 |

评分参照期限：**过去6个月**（特别注明者除外）

### 社会能力评分

社会能力各方面根据活动种类、参与程度等评定，计算社会能力总分。

### 结果判断（T分转换）

各因子粗分通过常模转换为标准分（T分）：

**行为问题因子判断：**
| T分范围 | 说明 |
|---------|-----|
| **T < 63** | 正常范围 |
| **T = 63-70** | 临界范围，需要关注 |
| **T > 70** | 异常范围，建议进一步评估 |

**宽带因子（内化/外化）及总行为问题判断：**
| T分范围 | 说明 |
|---------|-----|
| **T < 60** | 正常范围 |
| **T = 60-63** | 临界范围 |
| **T > 63** | 异常范围 |

**社会能力因子判断（分越低越有问题）：**
| T分范围 | 说明 |
|---------|-----|
| **T > 37** | 正常范围 |
| **T = 30-37** | 临界范围 |
| **T < 30** | 异常范围 |

## 注意事项

1. CBCL 主要由**父母或主要照料者**填写，反映家庭中的行为表现
2. 建议同时使用**教师报告量表（TRF）**以获取学校行为信息
3. 量表需参照**中国常模**进行T分转换，直接使用原始分无法判断结果
4. 量表为综合筛查工具，不能单独作为任何诊断的依据
5. **涉及自我伤害和自杀题目需特别关注**
