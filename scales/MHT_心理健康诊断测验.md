# MHT 心理健康诊断测验（中小学生版）

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 心理健康诊断测验（Mental Health Test） |
| 编制者 | 华东师范大学 周步成教授（根据日本铃木清等编制的"不安倾向诊断测验"修订） |
| 适用对象 | 小学四年级至高中阶段学生（约10-18岁） |
| 题目数量 | 100题 |
| 答题形式 | "是/否"二元选择 |
| 测试时间 | 约20-30分钟 |
| 用途 | 综合评定中小学生心理健康状况 |

---

### L 效度量表（测谎量表，8题）
这8道题分散于全卷中，均为社会赞许性陈述，真实作答应答"否"。若答"是"超过7题，该份答卷视为无效。

## 评分标准

### 答题计分
- 答"**是**"计 **1分**
- 答"**否**"计 **0分**

> 注：部分题目为反向计分（答"否"才是心理健康的表现），具体以测验手册为准。

### 效度量表判断
| 效度分 | 判断 |
|-------|------|
| ≤ 7分 | 答卷有效，可继续分析 |
| > 7分 | 答卷无效，不予分析（作答不真实） |

### 各内容量表解释
| 分数范围 | 心理状况解释 |
|---------|------------|
| **0-3分** | 心理健康状况良好，无明显焦虑倾向 |
| **4-7分** | 存在一定倾向，需要关注 |
| **≥ 8分** | 该方面存在较明显问题，建议重点辅导 |

### 全量表总分解释
| 总分 | 心理状况解释 | 建议处理 |
|-----|------------|--------|
| **< 45分** | 心理健康状况良好 | 常规关注 |
| **45-64分** | 有一定心理问题倾向 | 加强关注，做好疏导 |
| **≥ 65分** | 存在明显心理障碍 | 需制定特殊指导计划，建议专业干预 |

---

## 注意事项

1. 本测验仅为**筛查工具**，不能作为临床诊断依据
2. 结果需结合学生实际情况和日常表现综合判断
3. 发现高分学生应由专业心理教师进一步评估
4. 保护学生隐私，不可公开个别测验结果
