# 家庭功能与亲子关系量表系列

---

## 一、家庭亲密度与适应性量表（FACES II-CV）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 家庭亲密度与适应性量表第二版中文版（Family Adaptability and Cohesion Evaluation Scales） |
| 缩写 | FACES II-CV |
| 原版编制 | Olson DH等（1982年） |
| 中文版 | 费立鹏等修订（1991年） |
| 适用对象 | 中小学生（10岁以上，与家人共同填写） |
| 题目数量 | 30题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估家庭功能中的亲密性和灵活性 |

### 量表结构（2个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 亲密度（凝聚力） | 16题 | 家庭成员间的情感联结 |
| 适应性 | 14题 | 家庭面对压力时的灵活调整能力 |

### 代表性题目

| 维度 | 题目示例 |
|-----|---------|
| 亲密度 | 家庭成员之间互相关心 |
| 亲密度 | 家庭成员相互支持 |
| 亲密度 | 我们家庭成员之间关系亲密 |
| 适应性 | 家庭中有明确的规矩 |
| 适应性 | 家规是公平的 |
| 适应性 | 在家里，出了问题，大家愿意共同商量解决 |

### 评分标准

- 五级评分：1=不是，5=总是
- **亲密度总分**：16题得分之和（16-80分）
- **适应性总分**：14题得分之和（14-70分）

| 维度 | 分数范围 | 家庭类型 |
|-----|---------|---------|
| 亲密度低 | 亲密度 < 35 | 松散型家庭 |
| 亲密度中 | 35-65 | 连接型或亲密型 |
| 亲密度高 | > 65 | 缠结型家庭 |
| 适应性低 | 适应性 < 30 | 僵化型家庭 |
| 适应性中 | 30-50 | 有结构型或灵活型 |
| 适应性高 | > 50 | 混乱型家庭 |

## 二、父母教养方式量表（EMBU）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 父母教养方式评价量表（Egna Minnen Betraffande Uppfostran - My Memories of Upbringing） |
| 缩写 | EMBU |
| 原版编制 | Perris C等（1980年，瑞典） |
| 中文版 | 蒋奖等修订（1993年） |
| 适用对象 | 初中生至成人（12岁以上，回忆父母教养方式） |
| 题目数量 | 66题（父亲版）+ 66题（母亲版） |
| 答题形式 | 四级评分 |
| 测试时间 | 约20分钟 |
| 用途 | 评估父母的教养方式对心理发展的影响 |

### 量表结构（6个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 情感温暖与理解 | 19题 | 父母对子女的关爱和接纳 |
| 拒绝 | 12题 | 父母对子女行为的拒绝态度 |
| 过度保护 | 16题 | 过分关注和保护 |
| 惩罚与严厉 | 11题 | 严格管教和惩罚 |
| 过干涉 | 8题 | 过分介入子女的私生活 |
| 偏爱被试 | 6题 | 父母对被试的特别偏爱 |

### 代表性题目

| 维度 | 题目示例 |
|-----|---------|
| 情感温暖 | 父/母亲总是倾听我的意见 |
| 情感温暖 | 父/母亲尊重我的意见 |
| 拒绝 | 父/母亲经常对我感到失望 |
| 过度保护 | 父/母亲不允许我独立完成事情 |
| 惩罚 | 父/母亲会体罚我 |

### 评分标准

- 四级评分：1=从不，2=偶尔，3=经常，4=总是
- 各维度得分 = 该维度所有题目之和

## 三、家庭环境量表（FES-CV）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 家庭环境量表中文版（Family Environment Scale） |
| 缩写 | FES-CV |
| 编制者 | Moos RH & Moos BS（原版），费立鹏等修订中文版 |
| 适用对象 | 小学高年级至成人（10岁以上） |
| 题目数量 | 90题 |
| 答题形式 | 是/否二元选择 |
| 测试时间 | 约20-25分钟 |
| 用途 | 评估家庭社会环境特征 |

### 量表结构（10个分量表）

| 分量表 | 内容 |
|-------|-----|
| 亲密度 | 家庭成员间互相帮助和支持 |
| 情感表达 | 公开表达情感的鼓励程度 |
| 矛盾性 | 家庭成员间公开表达愤怒的程度 |
| 独立性 | 家庭成员的自主程度 |
| 成功性 | 将活动变为成就的程度 |
| 知识性 | 对政治、文化等的兴趣 |
| 娱乐性 | 参与娱乐活动的程度 |
| 道德宗教 | 对道德和宗教的强调程度 |
| 组织性 | 家庭活动的组织程度 |
| 控制性 | 家庭规则和程序的程度 |

### 评分标准

- 每题是=1，否=0（根据各分量表标准答案）
- 各分量表满分9分
- 分数越高，该方面特征越明显
