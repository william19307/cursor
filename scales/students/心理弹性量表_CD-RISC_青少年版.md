# 心理弹性（韧性）量表系列

---

## 一、Connor-Davidson 心理弹性量表（CD-RISC）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Connor-Davidson Resilience Scale |
| 缩写 | CD-RISC |
| 编制者 | Connor KM & Davidson JRT（2003年） |
| 中文版 | 于肖楠、张建新（2007年）修订 |
| 适用对象 | 初中生至成人（12岁以上） |
| 题目数量 | 25题 |
| 答题形式 | 五级评分（0-4分） |
| 测试时间 | 约10分钟 |
| 用途 | 评估个体面对逆境和压力的心理复原能力 |

### 完整题目（25题）

> 请根据**过去一个月**内您的感受，选择最符合的描述。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我能适应变化 |
| 2 | 我与朋友或家人有亲密的关系 |
| 3 | 有时上天或神明会帮助我 |
| 4 | 我能处理任何发生的事情 |
| 5 | 以前成功的经历帮助我面对新挑战 |
| 6 | 当事情看起来没有希望时，我不会轻易放弃 |
| 7 | 我能处理痛苦的感觉 |
| 8 | 有时命运或上天对我是有利的 |
| 9 | 我能从失败中爬起来 |
| 10 | 我认为自己是个坚强的人 |
| 11 | 我能做出不受欢迎或困难的决定 |
| 12 | 我能处理突如其来的状况 |
| 13 | 我能在压力下保持清醒 |
| 14 | 我做事有自信 |
| 15 | 我喜欢接受挑战 |
| 16 | 我在完成目标方面能付出足够努力 |
| 17 | 我对自己的成败坚持不懈 |
| 18 | 我能从批评中走出来 |
| 19 | 我能容忍不舒适的感受 |
| 20 | 我面对困难时会采取行动 |
| 21 | 我为自己的成就感到自豪 |
| 22 | 我感到自己能掌控生活中的事情 |
| 23 | 我能独立解决问题 |
| 24 | 我努力实现我的目标 |
| 25 | 我能从挫折中坚持下来 |

### 评分标准

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 0 | 完全不是这样 | 0分 |
| 1 | 很少是这样 | 1分 |
| 2 | 有时是这样 | 2分 |
| 3 | 经常是这样 | 3分 |
| 4 | 几乎总是这样 | 4分 |

**总分** = 25题得分之和（满分100分）

| 总分范围 | 心理弹性水平 |
|---------|-----------|
| 0-50分 | 低心理弹性，面对逆境能力较弱 |
| 51-70分 | 中等心理弹性 |
| 71-100分 | 高心理弹性，较强的复原力 |

**信度**：内部一致性 α = 0.89；重测信度 = 0.87

## 二、青少年心理韧性量表（RSCA）

> **来源**：胡月琴、甘怡群（2008年）编制，发表于《心理学报》第40卷第8期。以下27题来自公开学术网站。

### 基本信息

| 属性 | 内容 |
|------|------|
| 适用对象 | 中学生（12-18岁）|
| 题目数量 | 27题 |
| 答题形式 | 五级评分 |
| 信度 | 各因子 α > 0.70；整体 α = 0.85 |

### 量表结构（5个维度）

| 维度 | 归属 | 对应题号 |
|-----|-----|---------|
| 目标专注 | 个人力 | 1-5 |
| 情绪控制 | 个人力 | 6-11 |
| 积极认知 | 个人力 | 12-15 |
| 家庭支持 | 支持力 | 16-21 |
| 人际协助 | 支持力 | 22-27 |

### 完整题目（27题）

**目标专注（第1-5题）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我的生活有明确的目标 |
| 2 | 经历挫折后我一般会更加成熟有经验 |
| 3 | 面临困难，我一般会定一个计划和解决方案 |
| 4 | 面对困难时，我会集中自己的全部精力 |
| 5 | 我会为自己设定目标，以推动自己前进 |

**情绪控制（第6-11题）**

| 题号 | 题目内容 |
|-----|---------|
| 6 | 失败不会让我感到气馁 |
| 7 | 我能控制自己的不愉快情绪 |
| 8 | 失败和挫折不会让我怀疑自己的能力 |
| 9 | 我很快就能忘记不愉快的事情 |
| 10 | 我能在短时间内调整情绪 |
| 11 | 我情绪波动很小，不容易大起大落 |

**积极认知（第12-15题）**

| 题号 | 题目内容 |
|-----|---------|
| 12 | 我觉得与结果相比，事情的过程更能够帮助人成长 |
| 13 | 我认为逆境对人有激励作用 |
| 14 | 逆境有时候是对成长的一种帮助 |
| 15 | 我觉得任何事情都有其积极的一面 |

**家庭支持（第16-21题）**

| 题号 | 题目内容 |
|-----|---------|
| 16 | 父母尊重我的意见 |
| 17 | 父母不会干涉我的想法 |
| 18 | 在家里，我说什么总有人听 |
| 19 | 父母对我给予信心和精神上的支持 |
| 20 | 父母鼓励我全力以赴 |
| 21 | 父母很少苛责我 |

**人际协助（第22-27题）**

| 题号 | 题目内容 |
|-----|---------|
| 22 | 当我遇到不愉快的事情时，总能找到合适的倾诉对象 |
| 23 | 我有一个同龄朋友，可以把我的困难讲给他/她听 |
| 24 | 我遇到困难需要帮助时，我知道该去找谁 |
| 25 | 我习惯把事情向人倾诉 |
| 26 | 我有困难的时候会主动找别人倾诉 |
| 27 | 心情不好愿意跟别人说 |

### 评分标准

- 五级评分：1=完全不符合，2=比较不符合，3=说不清，4=比较符合，5=完全符合
- **个人力得分** = 第1-15题之和（15-75分）
- **支持力得分** = 第16-27题之和（12-60分）
- 总分 = 27题之和，分数越高心理韧性越强
