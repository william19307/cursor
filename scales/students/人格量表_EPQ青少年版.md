# EPQ 艾森克人格问卷（少年版）

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 艾森克人格问卷青少年版（Eysenck Personality Questionnaire-Junior） |
| 缩写 | EPQ-J |
| 编制者 | H.J. Eysenck & S.B.G. Eysenck（1975年） |
| 中文版 | 龚耀先修订（1983年）/ 陈仲庚修订版 |
| 适用对象 | 小学高年级至高中（7-15岁为主） |
| 题目数量 | 88题（简版48题） |
| 答题形式 | 是/否二元选择 |
| 测试时间 | 约15-20分钟 |
| 用途 | 评估青少年基本人格特征 |

---

## 量表结构（4个量表）

| 量表代码 | 量表名称 | 题目数 | 主要评估内容 |
|---------|---------|-------|------------|
| **E** | 内外向 | 21题 | 社交性、活力、冒险性 |
| **N** | 神经质 | 24题 | 情绪稳定性、焦虑倾向 |
| **P** | 精神质 | 23题 | 攻击性、冷漠度、不顺从 |
| **L** | 掩饰性 | 20题 | 答题真实性（测谎） |

---

## 代表性题目（各量表举例）

### E 量表（内外向）

| 题目示例 |
|---------|
| 你喜欢周围有许多使你高兴的事情吗？（是=外向） |
| 你爱聚会吗？（是=外向） |
| 你更喜欢安静地独处而不是参加晚会吗？（否=外向） |
| 你很容易和人成为好朋友吗？（是=外向） |
| 你爱说话吗？（是=外向） |

### N 量表（神经质）

| 题目示例 |
|---------|
| 你是否会感到莫名其妙的悲伤？（是=N高） |
| 你的情绪会有起伏吗？（是=N高） |
| 你常常感到孤独吗？（是=N高） |
| 你是个容易动情的人吗？（是=N高） |
| 你担忧的事情多吗？（是=N高） |

### P 量表（精神质）

| 题目示例 |
|---------|
| 如果不得不伤害某人，你会否忧虑吗？（否=P高） |
| 你喜欢结交陌生人吗？（否=P高） |
| 你喜欢捉弄别人吗？（是=P高） |
| 你是否觉得别人对你太过提防吗？（是=P高） |

### L 量表（掩饰性）

| 题目示例 |
|---------|
| 你曾经贪图过某些你没有权利得到的东西吗？（否=高L，回避社会不赞许行为） |
| 你曾取笑过别人的脏笑话吗？（否=高L） |

## 评分标准

### 计分方法

- 每题答"是"或"否"
- 各量表根据标准答案计1分（具体答案见量表手册）
- **各量表原始分满分**：E=21分，N=24分，P=23分，L=20分

### L 量表效度判断

| L 分 | 说明 |
|-----|-----|
| ≤ 12分（T < 60） | 答卷真实有效 |
| > 12分（T > 60） | 答卷可能存在掩饰，结果参考性下降 |

### T 分标准（参考常模转换）

| T分范围 | 解释 |
|--------|-----|
| > 61.5 | 典型（外向/神经质不稳定/精神质明显） |
| 56.7-61.5 | 趋向某一端 |
| 43.3-56.7 | 中间型 |
| 38.5-43.3 | 趋向另一端 |
| < 38.5 | 典型另一端（内向/情绪稳定） |

### E 量表解释

| E-T分 | 人格倾向 | 特点 |
|-------|---------|-----|
| > 61.5 | 典型外向 | 活泼开朗、渴望刺激、社交性强 |
| 43.3-61.4 | 中间型或倾向外向 | 兼有内外向特点 |
| < 43.3 | 典型内向 | 安静含蓄、喜独处、自控力强 |

### N 量表解释

| N-T分 | 人格倾向 | 特点 |
|-------|---------|-----|
| > 61.5 | 情绪不稳定 | 焦虑、抑郁、紧张、过于担心 |
| 43.3-61.4 | 中间型 | 情绪基本稳定 |
| < 43.3 | 情绪稳定 | 平静、放松、有自控力 |

### P 量表解释

| P-T分 | 人格倾向 | 特点 |
|-------|---------|-----|
| > 61.5 | 精神质明显 | 冷漠、孤独、不关心他人 |
| < 56.7 | 正常范围 | 正常人格 |

---

## 注意事项

1. EPQ 是人格特征描述工具，**不存在好坏之分**
2. L 分过高（> 60）提示结果可靠性差，应谨慎解读
3. P 分高在极端情况下需结合临床判断
4. 青少年版（EPQ-J）与成人版（EPQ-A）常模不同，不可混用
