# 教师职业认同与工作满意度量表系列

---

## 一、中小学教师职业认同量表（魏淑华版）

> **来源**：魏淑华等（2013年）编制，发表于《教师教育研究》第1期。以下结构信息来自该论文，完整题目需查阅原文（论文DOI或数据库：中国知网 CNKI）。

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 魏淑华等 |
| 发表来源 | 《教师教育研究》2013年第1期 |
| 适用对象 | 中小学教师 |
| 量表结构 | **4个因子**（注：早期版本有2-5因子不等，以2013年正式版为准）|
| 答题形式 | 五级评分（1=完全不符合，5=完全符合） |

### 四因子结构

| 因子 | 含义 |
|-----|-----|
| 职业价值观 | 对教师职业本身价值和意义的认可程度 |
| 角色价值观 | 对自身在教师角色中的价值评价 |
| 职业归属感 | 对教师群体的认同感和归属感 |
| 职业行为倾向 | 在职业活动中持续投入的意向 |

### 代表性题目示例（基于论文描述）

| 因子 | 题目示例 |
|-----|---------|
| 职业价值观 | 我认为教师职业是有意义的 |
| 职业价值观 | 我为从事教育工作感到自豪 |
| 角色价值观 | 我认为自己是一名有能力的教师 |
| 职业归属感 | 我有一种强烈的归属感，觉得自己是教师群体的一员 |
| 职业行为倾向 | 如果重新选择，我仍然会选择教师职业 |

> ⚠️ **说明**：完整题目需查阅原发表论文。上述为根据论文描述整理的示例题目，不代表原版逐字表述。

### 评分标准

- 五级评分：1=完全不符合，5=完全符合
- 各因子均分越高，职业认同感越强

## 二、教师工作满意度量表

> **来源**：基于Hackman & Oldham工作特征模型及中国本土化研究编制的量表，多个版本在学术文献中使用，以下为常用结构框架。

### 基本信息

| 属性 | 内容 |
|------|------|
| 适用对象 | 中小学教师 |
| 量表结构 | 6个维度，约27-30题 |
| 答题形式 | 五级评分（1=非常不满意，5=非常满意） |
| 用途 | 评估教师对工作各方面的满意程度 |

### 六个维度

| 维度 | 评估内容 |
|-----|---------|
| 工作本身 | 对教学工作内容和任务本身的满意度 |
| 薪酬福利 | 对工资、绩效、福利待遇的满意度 |
| 发展机会 | 对培训进修和晋升机会的满意度 |
| 学校管理 | 对学校领导和管理方式的满意度 |
| 人际关系 | 对同事合作和学生关系的满意度 |
| 工作环境 | 对办公条件和校园环境的满意度 |

### 代表性题目（各维度示例）

| 维度 | 题目示例 |
|-----|---------|
| 工作本身 | 我喜欢我的教学工作 |
| 工作本身 | 我的工作让我感到有挑战性和成就感 |
| 薪酬福利 | 我对目前的工资水平感到满意 |
| 薪酬福利 | 学校的福利待遇是公平合理的 |
| 发展机会 | 学校为我提供了足够的专业发展机会 |
| 发展机会 | 晋升制度是公平透明的 |
| 学校管理 | 学校领导对我的工作表示认可 |
| 学校管理 | 学校的各项制度是合理的 |
| 人际关系 | 我与同事相处融洽 |
| 工作环境 | 我的工作场所条件良好 |

### 评分标准

- 五级评分：1=非常不满意，5=非常满意
- 各维度均分解释：
  - 1-2分：不满意
  - 2-3分：一般
  - 3-4分：较满意
  - 4-5分：非常满意

> ⚠️ **说明**：教师工作满意度量表存在多个版本，上述为通用结构框架。如需标准化量表，建议参考文献：
> - 许燕（1999）. 中学教师工作满意度的调查研究. 北京师范大学学报
> - 刘晓明等（2004）. 中小学教师工作满意度量表的编制. 中国教育学刊
