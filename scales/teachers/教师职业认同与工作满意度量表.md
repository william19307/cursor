# 教师职业认同与工作满意度量表系列

---

## 一、教师职业认同量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 魏淑华等（2009年，中国本土编制） |
| 适用对象 | 中小学教师 |
| 题目数量 | 30题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估教师对教师职业的认同感和价值感 |

### 量表结构（5个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 职业价值观 | 7题 | 对教师职业价值和意义的认可 |
| 职业意志 | 5题 | 坚持从事教师职业的意志力 |
| 职业能力 | 7题 | 对自身教学能力的认可和信心 |
| 职业行为倾向 | 5题 | 对教学工作投入的行为意向 |
| 职业归属感 | 6题 | 对教师群体的归属和认同感 |

### 完整题目（30题）

> 请根据您的实际情况，选择最符合的选项。

**职业价值观（7题）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我认为教师是一份有价值的工作 |
| 2 | 教书育人是一项非常有意义的事业 |
| 3 | 我为从事教育工作而感到自豪 |
| 4 | 我相信优秀的教师能改变学生的命运 |
| 5 | 教师对社会的贡献是无可替代的 |
| 6 | 我愿意把自己的一生献给教育事业 |
| 7 | 我认为教育工作是实现自我价值的重要途径 |

**职业意志（5题）**

| 题号 | 题目内容 |
|-----|---------|
| 8 | 即使工作遇到困难，我也不会放弃教师职业 |
| 9 | 无论遇到什么挑战，我都会坚持做一名好教师 |
| 10 | 我打算在教育岗位上一直工作下去 |
| 11 | 当教学遇到挫折时，我仍然能够坚持 |
| 12 | 我对从事教师职业有坚定的信念 |

**职业能力（7题）**

| 题号 | 题目内容 |
|-----|---------|
| 13 | 我认为自己是一名有能力的教师 |
| 14 | 我相信自己能够处理好课堂中的各种问题 |
| 15 | 我的教学方法是有效的 |
| 16 | 我能够胜任我目前的教学工作 |
| 17 | 我持续提升自己的专业能力 |
| 18 | 我能够帮助学生解决他们的学习困难 |
| 19 | 我认为自己是一名专业的教育工作者 |

**职业行为倾向（5题）**

| 题号 | 题目内容 |
|-----|---------|
| 20 | 我会主动参与教研活动来提升自己 |
| 21 | 我愿意花时间备好每一堂课 |
| 22 | 我愿意花时间与学生沟通交流 |
| 23 | 我积极参与学校的各类活动 |
| 24 | 我愿意学习新的教育理念和方法 |

**职业归属感（6题）**

| 题号 | 题目内容 |
|-----|---------|
| 25 | 我为成为一名教师而感到高兴 |
| 26 | 我认同教师这一群体的价值观 |
| 27 | 我愿意与其他教师分享经验和资源 |
| 28 | 我与同事之间有共同的职业情感 |
| 29 | 学校是我的第二个家 |
| 30 | 我与学生之间有深厚的情感联结 |

### 评分标准

- 五级评分：1=完全不符合，5=完全符合
- 各维度均分 = 该维度得分之和 ÷ 题目数
- 总分越高，职业认同感越强

## 二、教师工作满意度量表（TJSQ）

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 邱秀芳等（国内本土编制） |
| 适用对象 | 中小学教师 |
| 题目数量 | 27题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估教师对工作各方面的满意程度 |

### 量表结构（6个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 工作本身满意度 | 5题 | 对教学工作内容的满意程度 |
| 薪酬满意度 | 4题 | 对工资、奖金、福利的满意程度 |
| 发展机会满意度 | 5题 | 对培训和晋升机会的满意程度 |
| 管理满意度 | 5题 | 对学校领导和管理的满意程度 |
| 同事关系满意度 | 4题 | 对同事合作关系的满意程度 |
| 工作环境满意度 | 4题 | 对工作条件和环境的满意程度 |

### 代表性题目

| 维度 | 题目示例 |
|-----|---------|
| 工作本身 | 我喜欢我的教学工作 |
| 工作本身 | 我的工作对我有挑战性，让我有成就感 |
| 薪酬 | 我对目前的工资水平感到满意 |
| 发展机会 | 学校为我提供了足够的专业发展机会 |
| 管理 | 学校领导对我的工作表示认可 |
| 同事关系 | 我与同事相处融洽 |
| 工作环境 | 我的工作场所干净、安全、舒适 |

### 评分标准

- 五级评分：1=非常不满意，5=非常满意
- 各维度均分 = 该维度得分之和 ÷ 题目数
- 总均分解释：
  - 1-2分：不满意
  - 2-3分：一般
  - 3-4分：较满意
  - 4-5分：非常满意
