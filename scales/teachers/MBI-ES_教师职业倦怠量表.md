# MBI-ES 教师职业倦怠量表

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 马斯拉奇职业倦怠量表教育版（Maslach Burnout Inventory-Educators Survey） |
| 缩写 | MBI-ES |
| 编制者 | Maslach C & Jackson SE（1981年）；Maslach C等（1996年，教育版） |
| 中文版 | 伍新春等修订（2000年），中国标准化 |
| 适用对象 | 中小学教师、幼儿园教师 |
| 题目数量 | 22题 |
| 答题形式 | 六级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估教师职业倦怠的程度 |

---

## 量表结构（3个维度）

| 维度 | 题目数 | 主要内容 | 典型感受 |
|-----|-------|---------|---------|
| 情绪衰竭（EE） | 9题 | 情绪资源耗尽，极度疲竭 | "我筋疲力尽了" |
| 非人性化（DP） | 5题 | 对学生的冷漠和疏远 | "我不在乎学生了" |
| 低成就感（PA） | 8题 | 对工作意义和价值的消极评价 | "我做不好这份工作" |

---

## 评分标准

### 各题评分（六级评分）

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 1 | 非常不赞同 | 1分 |
| 2 | 不赞同 | 2分 |
| 3 | 有些不赞同 | 3分 |
| 4 | 有些赞同 | 4分 |
| 5 | 赞同 | 5分 |
| 6 | 非常赞同 | 6分 |

> 注：低成就感维度（第15-22题）均为反向计分（该维度分数越低，低成就感越强）

### 各维度计分方式

- **EE（情绪衰竭）**：第1-9题得分之和（9-54分），分数越高倦怠越重
- **DP（非人性化）**：第10-14题得分之和（5-30分），分数越高倦怠越重
- **PA（个人成就感）**：第15-22题得分之和（8-48分），分数越低倦怠越重（注意方向）

### 职业倦怠判断标准

#### 方案一（各维度独立判断）

| 维度 | 低倦怠 | 中度倦怠 | 高倦怠 |
|-----|-------|---------|-------|
| 情绪衰竭（EE） | ≤ 16分 | 17-26分 | ≥ 27分 |
| 非人性化（DP） | ≤ 6分 | 7-12分 | ≥ 13分 |
| 个人成就感（PA，高分=好） | ≥ 39分 | 32-38分 | ≤ 31分 |

#### 方案二（总体判断）

| 判断条件 | 倦怠级别 |
|---------|---------|
| 三个维度均处于低倦怠 | 无倦怠 |
| 一个或多个维度处于中度倦怠 | 轻度职业倦怠 |
| 一个维度处于高倦怠 | 中度职业倦怠 |
| 两个或三个维度处于高倦怠 | 高度职业倦怠 |

| 指标 | EE | DP | PA |
|-----|---|---|---|
| Cronbach's α | 0.90 | 0.75 | 0.82 |
| 重测信度 | 0.69 | 0.56 | 0.61 |

---

## 注意事项

1. MBI-ES 是目前国际上应用最广泛的职业倦怠测量工具
2. 三个维度应独立解读，不建议简单加总
3. 结果受工作年限、学段、任教科目等因素影响，解读时需考虑背景
4. 高倦怠教师需要管理层和机构层面的支持与干预
