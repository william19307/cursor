# 中小学教师工作压力量表

> **来源**：徐长江等编制，相关研究发表于《教育研究》1999年及后续学术论文。以下结构框架来自公开学术文献，完整53题原始版本需查阅作者原文。

## 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 徐长江等 |
| 适用对象 | 中小学教师 |
| 题目数量 | 53题（压力源36题 + 压力反应17题） |
| 答题形式 | 五级评分 |
| 测试时间 | 约15-20分钟 |
| 用途 | 全面评估教师工作压力来源和压力反应 |

---

## 第一部分：工作压力源（36题）

> 请评估以下各事项对您造成压力的程度。

### 维度一：教学与评估压力（约8题）

| 示例题目 |
|---------|
| 领导随机听课或专家评估课 |
| 承担公开课、示范课的教学任务 |
| 进行教学成绩的评比和排名 |
| 跟上新课程改革的要求和变化 |
| 课程内容多、教学进度紧张 |
| 命题、阅卷等考试相关工作 |
| 课堂教学效果达不到预期 |
| 将现代教育技术应用于教学 |

### 维度二：工作负担压力（约8题）

| 示例题目 |
|---------|
| 工作量过大，时间不够用 |
| 行政事务性工作（填表、写总结等）太多 |
| 要开各类会议，影响正常教学 |
| 班级人数过多，难以顾及每位学生 |
| 课外辅导学生的时间和精力 |
| 作业批改量大，占用大量时间 |
| 在规定时间内完成多项任务的压力 |
| 无暇照顾家庭和个人生活 |

### 维度三：职业发展压力（约6题）

| 示例题目 |
|---------|
| 担心自己的知识技能跟不上时代 |
| 专业发展和继续教育的要求 |
| 职称晋升竞争激烈 |
| 缺乏有经验教师的指导和帮助 |
| 对自己的职业发展前景感到迷茫 |
| 学校提供的培训机会不足 |

### 维度四：学生管理压力（约6题）

| 示例题目 |
|---------|
| 管理学习态度不好的学生 |
| 处理课堂纪律问题 |
| 学生个体差异大，难以因材施教 |
| 面对特殊行为或问题学生 |
| 关注学生心理健康和情绪状态 |
| 处理学生之间的冲突或纠纷 |

### 维度五：组织与人际压力（约8题）

| 示例题目 |
|---------|
| 与领导之间关系的处理 |
| 同事间的竞争与不合作 |
| 学校管理方式和决策不够透明 |
| 与家长沟通和协调的压力 |
| 学校内部利益分配不公平 |
| 工作中缺乏自主权和话语权 |
| 社会对教师职业的负面评价 |
| 教师工资和福利待遇偏低 |

## 第二部分：工作压力反应（17题）

> 请评估您目前在以下方面出现这些反应的频率。

### 躯体反应（约5题）

| 示例题目 |
|---------|
| 我经常感到头痛、偏头痛 |
| 我容易疲劳，精力不足 |
| 我睡眠不好，经常失眠或噩梦 |
| 我时常感到身体某部位不舒服 |
| 我的食欲发生了明显变化 |

### 情绪反应（约5题）

| 示例题目 |
|---------|
| 我感到焦虑、紧张 |
| 我经常情绪低落、闷闷不乐 |
| 我容易发怒或焦躁 |
| 我对工作感到越来越厌倦 |
| 我对教学工作感到失望 |

### 认知反应（约4题）

| 示例题目 |
|---------|
| 我上课时注意力难以集中 |
| 我的记忆力下降，容易遗忘 |
| 做决定时我感到困难 |
| 我对工作失去了创造力和新鲜感 |

### 行为反应（约3题）

| 示例题目 |
|---------|
| 我开始回避某些工作任务 |
| 我的工作效率明显下降 |
| 我在课堂上越来越缺乏热情 |

## 评分标准

### 压力源评分（五级）

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 0 | 没有压力 | 0分 |
| 1 | 压力较轻 | 1分 |
| 2 | 压力中等 | 2分 |
| 3 | 压力较大 | 3分 |
| 4 | 压力很大 | 4分 |

### 压力反应评分（五级）

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 0 | 从来没有 | 0分 |
| 1 | 很少 | 1分 |
| 2 | 有时 | 2分 |
| 3 | 经常 | 3分 |
| 4 | 总是 | 4分 |

### 结果解释

- **压力源总分**（0-144分）：分数越高，压力来源越多
- **压力反应总分**（0-68分）：分数越高，身心反应越强烈

| 压力源总分 | 压力水平 |
|---------|---------|
| 0-36分 | 低压力 |
| 37-72分 | 中等压力 |
| 73-108分 | 较高压力 |
| 109-144分 | 极高压力，需要关注 |
