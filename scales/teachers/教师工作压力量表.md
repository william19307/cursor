# 中小学教师工作压力量表

## 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 徐长江等（中国本土编制） |
| 适用对象 | 中小学教师 |
| 题目数量 | 53题（压力源36题 + 压力反应17题） |
| 答题形式 | 五级评分 |
| 测试时间 | 约15-20分钟 |
| 用途 | 全面评估教师工作压力的来源和反应 |

---

## 量表结构

### 第一部分：工作压力源（36题）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 社会因素 | 8题 | 社会对教师的期望和评价 |
| 职业因素 | 8题 | 教学任务、专业发展 |
| 组织因素 | 8题 | 学校管理和人际关系 |
| 专业能力 | 6题 | 应对教学挑战的能力感 |
| 学生因素 | 6题 | 管理学生的困难 |

### 第二部分：压力反应（17题）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 躯体反应 | 5题 | 身体不适（头痛、失眠等） |
| 情绪反应 | 5题 | 焦虑、抑郁、易怒 |
| 认知反应 | 4题 | 注意力下降、决策困难 |
| 行为反应 | 3题 | 工作效率降低、回避 |

## 评分标准

### 压力源评分

| 选项 | 压力程度 | 得分 |
|-----|---------|-----|
| 1 | 没有压力 | 0分 |
| 2 | 压力较轻 | 1分 |
| 3 | 压力中等 | 2分 |
| 4 | 压力较大 | 3分 |
| 5 | 压力很大 | 4分 |

### 压力反应评分

| 选项 | 频率 | 得分 |
|-----|-----|-----|
| 1 | 从来没有 | 0分 |
| 2 | 很少 | 1分 |
| 3 | 有时 | 2分 |
| 4 | 经常 | 3分 |
| 5 | 总是 | 4分 |

### 结果解释

- **压力源总分** = 36题之和（0-144分），分数越高压力源越多
- **压力反应总分** = 17题之和（0-68分），分数越高反应越强烈

| 压力源总分 | 压力水平 |
|---------|---------|
| 0-36分 | 低压力 |
| 37-72分 | 中等压力 |
| 73-108分 | 较高压力 |
| 109-144分 | 极高压力 |
