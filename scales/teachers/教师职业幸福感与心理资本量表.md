# 教师职业幸福感与心理资本量表系列

---

## 一、教师职业幸福感量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 申继亮等（北京师范大学，中国本土） |
| 适用对象 | 中小学教师 |
| 题目数量 | 24题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估教师在职业中的幸福感体验 |

### 量表结构（4个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 工作意义感 | 6题 | 感受到工作的价值和重要性 |
| 工作快乐感 | 6题 | 工作中的积极情绪体验 |
| 工作成就感 | 6题 | 对工作成果的满足感 |
| 工作投入感 | 6题 | 对工作的专注和热情 |

### 完整题目（24题）

> 请根据您在工作中的实际感受，选择最符合的选项。

**工作意义感（6题）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我觉得我的工作对社会很有意义 |
| 2 | 我觉得教师是一份有价值的职业 |
| 3 | 我认为自己的工作对学生的成长很重要 |
| 4 | 我的工作让我感到生命的意义 |
| 5 | 教育工作让我觉得自己的生命是有目的的 |
| 6 | 从事教育让我感到充实 |

**工作快乐感（6题）**

| 题号 | 题目内容 |
|-----|---------|
| 7 | 我享受与学生一起的时光 |
| 8 | 我喜欢自己的工作 |
| 9 | 上课让我感到愉悦和满足 |
| 10 | 当我看到学生进步时，我感到由衷的喜悦 |
| 11 | 备课和教研让我感到充实 |
| 12 | 即使在困难的日子，工作也能给我带来快乐 |

**工作成就感（6题）**

| 题号 | 题目内容 |
|-----|---------|
| 13 | 我认为自己的教学是有效果的 |
| 14 | 看到学生取得进步让我感到有成就感 |
| 15 | 我为自己成功帮助学生解决问题而感到自豪 |
| 16 | 我完成了对自己工作的预期目标 |
| 17 | 我在职业发展上取得了令自己满意的成就 |
| 18 | 我受到同事和领导的认可 |

**工作投入感（6题）**

| 题号 | 题目内容 |
|-----|---------|
| 19 | 我会为教学工作全力以赴 |
| 20 | 我愿意花额外的时间做好本职工作 |
| 21 | 即使没有要求，我也会主动提升自己 |
| 22 | 我把工作当作自己的事业来对待 |
| 23 | 我的工作热情不减 |
| 24 | 我乐于在工作上进行创新尝试 |

### 评分标准

- 五级评分：1=完全不符合，5=完全符合
- 各维度得分 = 该维度6题之和（6-30分）
- 总分范围：24-120分，分数越高幸福感越强

## 二、教师心理资本量表（PCQ-24）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 心理资本量表教师版（Psychological Capital Questionnaire for Teachers） |
| 参考编制 | 基于Luthans等PCQ量表（2007年）的教师版本土化 |
| 适用对象 | 中小学教师 |
| 题目数量 | 24题 |
| 答题形式 | 六级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估教师的积极心理资本状态 |

### 量表结构（HERO模型，4个维度）

| 维度 | 缩写 | 题目数 | 主要内容 |
|-----|-----|-------|---------|
| 希望 | H | 6题 | 对目标的意志力和实现路径 |
| 效能感 | E | 6题 | 对完成工作任务的信心 |
| 韧性 | R | 6题 | 从挫折和挑战中恢复的能力 |
| 乐观 | O | 6题 | 对当前和未来结果的积极归因 |

### 完整题目（24题）

**希望（6题）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我对自己的职业目标充满信心 |
| 2 | 如果被逼到绝路，我能想到很多方法走出困境 |
| 3 | 我能独立思考问题的解决方案 |
| 4 | 此刻，我正在朝着自己的工作目标努力 |
| 5 | 我认为自己非常成功 |
| 6 | 我能想到很多实现工作目标的方法 |

**效能感（6题）**

| 题号 | 题目内容 |
|-----|---------|
| 7 | 我有信心向学生讲解课程内容 |
| 8 | 我有信心代表学校在会议上做报告 |
| 9 | 我有信心帮助制定学校的教学策略 |
| 10 | 我有信心联系与工作相关的专业领域的人士 |
| 11 | 我有信心向学生家长解释自己的教学方法 |
| 12 | 我有信心为解决工作中的难题做出贡献 |

**韧性（6题）**

| 题号 | 题目内容 |
|-----|---------|
| 13 | 在工作中，我能处理很多任务 |
| 14 | 在工作中遇到挫折时，我可以很快恢复 |
| 15 | 我能同时处理多项工作任务 |
| 16 | 我不会为工作上的困难感到沮丧 |
| 17 | 我以前克服过很多困难 |
| 18 | 在学校里，我一般能处理任何发生的事情 |

**乐观（6题）**

| 题号 | 题目内容 |
|-----|---------|
| 19 | 在工作中，很多事情都向好的方向发展 |
| 20 | 关于工作，我总是把好的一面放在前面 |
| 21 | 在工作中，我总认为事情会往好处想 |
| 22 | 我对工作上将要发生的事情感到乐观 |
| 23 | 事情不确定时，我通常期望最好的结果 |
| 24 | 我对于自己工作的愉快程度充满期待 |

### 评分标准

- 六级评分：1=强烈不同意，6=强烈同意
- 各维度得分 = 6题之和（6-36分）
- 心理资本总分 = 4个维度之和（24-144分），分数越高心理资本越丰富

## 三、教师工作投入量表（UWES）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Utrecht工作投入量表（Utrecht Work Engagement Scale） |
| 缩写 | UWES |
| 编制者 | Schaufeli WB等（2002年） |
| 适用对象 | 教师及各职业人群 |
| 题目数量 | 17题（简版9题） |
| 答题形式 | 七级评分 |
| 用途 | 评估对工作的活力、奉献和专注程度 |

### 量表结构（3个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 活力（Vigor） | 6题 | 工作时的精力和韧性 |
| 奉献（Dedication） | 5题 | 对工作的投入感和意义感 |
| 专注（Absorption） | 6题 | 完全专注于工作的状态 |

### 代表性题目

| 维度 | 题目示例 |
|-----|---------|
| 活力 | 工作时，我感到精力充沛 |
| 活力 | 工作时，我感到自己强壮而有活力 |
| 奉献 | 我对自己的工作充满热情 |
| 奉献 | 我的工作给我鼓舞和激励 |
| 专注 | 工作时，我忘记了时间的流逝 |
| 专注 | 我沉浸在工作中时，感到非常快乐 |

### 评分标准

- 七级评分：0=从不，6=每天/总是
- 各维度均分越高，工作投入度越高
