## 四、快速评分速查

### 学生量表阳性阈值速查

| 量表 | 阳性判断标准 |
|-----|------------|
| MHT | 各分量表 ≥8分；总分 ≥65分 |
| MSSMHS | 任一分量表均分 ≥2.0 |
| SCARED | 总分 ≥23（8-11岁）/≥22-24（12-18岁） |
| CDI | ≥13（6-8岁）/≥17（9-12岁）/≥19（12+） |
| PHQ-9 | ≥10分 中度以上 |
| GAD-7 | ≥10分 中度以上 |
| SAS | 标准分 ≥50分 |
| SDS | 标准分 ≥50分 |
| SCL-90 | 总分 ≥160；任一因子均分 ≥2 |
| SDQ | 困难总分 ≥17分 |
| SES（自尊） | <20分 低自尊 |
| CD-RISC | <51分 低弹性 |
| IAT（网络成瘾） | ≥40分 有成瘾倾向 |
| PSS-10（压力） | ≥27分 高压力 |
| ASLEC | 参照常模比较 |

## 五、危机信号处理指引

以下量表中若出现阳性，须**立即**启动危机干预程序：

| 量表 | 危机题目 | 触发条件 |
|-----|---------|---------|
| CDI | 第9题（自杀意念） | 得分 ≥1分 |
| PHQ-9 | 第9题（自杀/自伤念头） | 得分 ≥1分 |
| SDS | 第19题（死亡想法） | 得分 ≥3分 |
| SCL-90 | 第15题（结束生命） | 得分 ≥3分 |
| CBCL | 第91题（谈论自杀） | 得分 ≥1分 |
| BSSI | 任意题目 | 得分 >0分 |
| C-SSRS | 3-5级意念 | 任何意念 |
| NSSI | 自伤行为存在 | 有过行为 |

**紧急联系**：全国心理援助热线 **400-161-9995** | 未成年人帮助热线 **12355**
