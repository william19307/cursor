## 快速选择指南

### 按学段选择

**小学（6-12岁）：**
- 综合筛查：MHT（四年级以上）
- 焦虑筛查：SCARED
- 抑郁筛查：CDI
- 行为问题：SDQ / CBCL（父母版）

**初中（12-15岁）：**
- 综合筛查：MSSMHS（MMHI-60）/ MHT
- 焦虑筛查：SCARED / SAS
- 抑郁筛查：PHQ-9 / CDI

**高中（15-18岁）：**
- 综合筛查：MSSMHS / SCL-90
- 焦虑筛查：GAD-7 / SAS
- 抑郁筛查：PHQ-9 / SDS

## 各量表评分速查

| 量表 | 评分方式 | 阳性/问题阈值 |
|-----|---------|-------------|
| MHT | 是/否（0-1分） | 各分量表 ≥8分；总分 ≥65分 |
| MSSMHS | 1-5分，取均分 | 任一分量表均分 ≥2.0分 |
| SCARED | 0-2分，求总分 | 总分≥23（8-11岁）；≥22/24（12-18岁） |
| CDI | 0-2分，求总分 | ≥13（6-8岁）；≥17（9-12岁）；≥19（12+） |
| PHQ-9 | 0-3分，满分27 | ≥10分：中度及以上抑郁 |
| GAD-7 | 0-3分，满分21 | ≥10分：中度及以上焦虑 |
| SAS | 粗分×1.25=标准分 | 标准分 ≥50分 |
| SDS | 粗分×1.25=标准分 | 标准分 ≥50分 |
| SCL-90 | 1-5分 | 总分 ≥160分；任一因子均分 ≥2分 |
| SDQ | 0-2分 | 困难总分 ≥17分 |

## 重要说明

> **所有量表均为筛查工具，不能作为临床诊断依据。**
> 筛查阳性结果需由专业心理工作者或精神科医生进行进一步评估。

- 发现学生有自杀/自伤想法时，需**立即**进行危机评估和干预
- 测验结果属个人隐私，严格保密
- 建议结合多来源信息（学生自评 + 家长 + 教师）综合判断
