## 评分速查

### 学生量表阳性阈值

| 量表 | 需关注的阈值 |
|-----|------------|
| MHT | 各分量表 ≥8分；总分 ≥65分 |
| MSSMHS | 任一分量表均分 ≥2.0 |
| SCL-90 | 总分 >160；任一因子均分 ≥2 |
| SCARED | ≥23分（8-11岁）/ ≥22-24分（12-18岁）|
| CDI | ≥13分（6-8岁）/ ≥17分（9-12岁）/ ≥19分（12+）|
| PHQ-9 | ≥10分 中度抑郁 |
| GAD-7 | ≥10分 中度焦虑 |
| SAS | 标准分 ≥50分 |
| SDS | 标准分 ≥50分 |
| SDQ | 困难总分 ≥17分 |
| CD-RISC | <51分 低弹性 |
| IAT | ≥40分 有成瘾倾向 |

### 教师量表阳性阈值

| 量表 | 需关注的阈值 |
|-----|------------|
| DASS-21（抑郁） | 原始分×2 ≥14分 |
| DASS-21（焦虑） | 原始分×2 ≥10分 |
| DASS-21（压力） | 原始分×2 ≥19分 |
| MBI-ES 情绪衰竭 | ≥27分 高倦怠 |
| MBI-ES 非人性化 | ≥13分 高倦怠 |
| MBI-ES 个人成就感 | ≤31分 低成就 |
