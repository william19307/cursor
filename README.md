## 评分速查

| 量表 | 判断标准 |
|-----|---------|
| MHT | 各分量表 ≥8分；总分 ≥65分 需干预 |
| MSSMHS | 任一分量表均分 ≥2.0 需关注 |
| SCARED | 总分 ≥23分（8-11岁）|
| CDI | ≥13/17/19分（按年龄段）|
| PHQ-9 | ≥10分 中度抑郁 |
| GAD-7 | ≥10分 中度焦虑 |
| SAS/SDS | 标准分 ≥50分 |
| SCL-90 | 总分 >160；任一因子 ≥2分 |
| MBI-ES（教师） | 情绪衰竭 ≥27分 高倦怠 |
| DASS-21（教师） | 抑郁换算分 ≥14；焦虑 ≥10；压力 ≥19 |
